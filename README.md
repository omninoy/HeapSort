# HeapSort
## Структура проекта

- **HeapSort/** - Серверное приложение
- **HeapSortClient/** - Клиентское консольное приложение
- **HeapSort.Test/** - Набор из 15 тестов
### 1. Запуск серверного приложения

```bash
cd HeapSort
dotnet run
```
### 2. Запуск клиентского приложения

В новом терминале:

```bash
cd HeapSortClient
dotnet run
```
### 3. Запуск Тестов

В новом терминале:

```bash
cd HeapSort.Test
dotnet test
```
